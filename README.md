# SysMonitor Pro

Система SysMonitor Pro выполняет всесторонний мониторинг ключевых компонентов компьютера в реальном времени, обеспечивая незаметную работу с минимальной нагрузкой (<2% CPU) и полной совместимостью с Windows 10/11 (x64).

Мониторинг аппаратных ресурсов:
- CPU/GPU: Загрузка по ядрам (%), частота (MHz), температура (°C) через WMI и OpenHardwareMonitor API; детализация топ-процессов.
- RAM: Использование (%), swap-файл, скорость аллокации/деаллокации; предупреждения при >90% загрузки.
- температура: Сенсоры материнской платы, HDD/SSD, видеокарты; графики трендов с алертами при критических значениях (>85°C).

Мониторинг дисковой подсистемы:
- пространство: свободно/занято (GB, %);
- IOPS/производительность: Скорость чтения/записи (MB/s), latency (мс), очередь запросов;

Сетевой мониторинг:
- трафик: In/Out (Mbps) по адаптерам, топ-порты/протоколы (TCP/UDP); суточные/недельные графики.
- пинг: Задержка (мс), потери пакетов до ключевых хостов (gateway, DNS); алерты при >200мс.
- порты: Активные соединения (netstat), обнаружение подозрительного трафика (DDoS, сканирование).
